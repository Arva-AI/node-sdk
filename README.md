# Arva NodeJS SDK

This repository contains the source code for the Arva AI API NodeJS SDK.

To install it, run:

```SH
npm i @arva-ai/sdk
```

To get started, create a client with:

```TS
const client = new Arva(
    'arva_xxxx' // your API key
)
```

Full docs [here](https://platform.arva-ai.com/docs).
